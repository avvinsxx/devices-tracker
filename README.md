# Device Tracker

Панель для отслеживания устройств на карте

## Стек

- Vue 3
- Pinia
- YMap

## Запуск

Для любого сценария установите зависимости

```sh
npm install
```

### Разработка

```sh
npm run dev
```

### Продакшен сборка

```sh
npm run build
```

### Безопасность

.env файл содержит тестовый ключ, так сделано осознано
